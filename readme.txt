Frontend developed using Visual Studio Code.  
Backend developed using IntelliJ IDEA.

To run the full project locally:

1. Clone the repository:
   git clone [REPO_LINK_HERE]

2. Running the Frontend:
   - Navigate to the `frontend/` directory
   - Open it in Visual Studio Code
   - Run the following commands:
       npm install
       npm run dev

3. Running the Backend:
   - Navigate to the `backend/` directory
   - Open it in IntelliJ IDEA
   - Run the main Spring Boot application

Important:
- Ensure that the PostgreSQL database has been created beforehand.
- The backend will automatically generate the required tables upon first run.
- The tables can be populated with data using SQL insert commands or external tools like pgAdmin.
